Echonym

A highly secure, offline-first anonymous social platform engineered for massive scale.

A comprehensive full-stack application prioritizing zero-latency UX and extreme performance. The frontend utilizes a robust 3-tier architecture (Riverpod → SQLite → Firestore), guaranteeing instantaneous startup via silent background FCM syncing and aggressive media caching. The Node.js/TypeScript backend employs an event-driven architecture with zero client-side database writes, enforcing all mutations through secure, transactional Cloud Functions.

  • Performance & Scalability: Eliminated N+1 Firestore queries via batched Riverpod providers. Implemented custom LRU memory/disk media caching and Optimistic UI.
  • Security & Moderation: Hardened infrastructure using AppCheck (Play Integrity), SHA-256 device fingerprinting, and automated Hive AI moderation pipelines for all uploads.
  • Audio DSP & Custom UI: Real-time voice anonymization using complex FFmpeg filter chains and 60fps mathematically generated waveform CustomPainters.
  • Backend Architecture: Event-driven Node.js backend using Firestore triggers for async data aggregation and hourly cron jobs for logarithmic trending decay.
  • Advanced Discovery: Custom geospatial bounding-box search, Algolia semantic search, and a multi-variable logarithmic group popularity ranking algorithm.

Genspire

Where startup builders find their next co-founder — not their next follower.

A social portfolio and live collaboration network for builders. Built with a recommendation engine on top of Gemini embeddings, real-time voice lounges, and an offline-first sync layer so the feed loads instantly even on bad connections.

  • Gemini (embedding-001) vector search matching builders to relevant projects
  • Multi-room live voice lounges via Agora RTC, with waitlists and auto FCM alerts
  • Cut Firestore reads by 70% with an in-memory name-caching layer
  • Offline-first architecture — SQLite cache loads the feed in ~50ms
